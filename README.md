# Anubis
3D Action RPG made in Bevy.

> [!WARNING]  
> This project is still in early development stage.

## Building
### Game
1. Install [Rust toolchain](https://rust-lang.org/learn/get-started/)
2. Install [Bevy dependencies](https://bevy.org/learn/quick-start/getting-started/setup/#installing-os-dependencies)
3. Run `cargo build` or `cargo run`

### Docs
1. Install [mdBook](https://rust-lang.github.io/mdBook/guide/installation.html)
2. Run `mdbook build docs` or `mdbook serve docs`
